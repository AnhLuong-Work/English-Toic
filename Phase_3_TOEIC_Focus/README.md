# Phase 3 — TOEIC Focus (Tuần 13–20)

## Mục tiêu giai đoạn này
- Từ vựng: 400 → **700 từ** (tổng tích lũy)
- Điểm TOEIC: 550 → **650+**
- Thành thạo chiến thuật từng Part
- Tốc độ đọc: 200+ từ/phút
- Tốc độ nghe: hiểu 80%+ nội dung

1. **Mỗi ngày tối thiểu 20 câu TOEIC thực** (Part 5 hoặc Part 7)

3. **Nghe mỗi ngày** — dù chỉ 15 phút BBC/VOA/TOEIC podcast
