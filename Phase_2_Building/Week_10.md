## Ngữ pháp — Câu Phức Nâng Cao

### Mệnh đề trạng ngữ nâng cao

| Mệnh đề | Từ nối | Ví dụ |
|---------|--------|-------|
| Thời gian | as soon as, by the time, once | **Once** the migration is complete, uptime will improve. |
| Nguyên nhân | given that, now that, seeing that | **Given that** cybersecurity risks are high, we upgraded the firewall. |
| Điều kiện | provided (that), assuming (that), on condition that | **Provided that** the API is stable, we'll proceed with deployment. |
| Mục đích | so that, in order that, lest | We encrypted the data **so that** it remains secure during migration. |
| Tương phản | whereas, while, even if | **Whereas** the old system was slow, the new SaaS solution is scalable. |

### Bài tập — Điền từ nối phù hợp:
1. ______ the data breach was contained quickly, no customer information was compromised.
2. We'll upgrade the legacy system ______ it can support the new IoT devices.
3. ______ the DevOps team deploys the patch, the bug will be resolved.
4. The algorithm needs to be optimized ______ latency doesn't affect user experience.
5. ______ Cloud A is cheaper, Cloud B offers better scalability.

*Đáp án: 1-Although/Even though, 2-so that, 3-Once/As soon as, 4-so that/lest, 5-Whereas/While*

## TOEIC Part 6 — Chiến thuật Chèn câu

### Câu hỏi chèn câu (Sentence Insertion)
Trong mỗi bài Part 6, có **1 câu hỏi** yêu cầu chọn câu phù hợp để chèn vào chỗ trống.

**Chiến thuật:**
1. Đọc câu **trước** và câu **sau** chỗ trống
2. Tìm từ khóa kết nối (therefore, however, in addition, for example...)
3. Kiểm tra đại từ (it, they, this, that) → phải tham chiếu rõ ràng
4. Kiểm tra thì động từ → phải nhất quán với ngữ cảnh

### Luyện tập Part 6 (toàn bài)

> To: IT Department  
> From: Sandra Miller, Operations Director  
> Subject: System Migration Update
>
> I am writing to provide an update on our planned migration from the legacy system to the new cloud computing platform. ______(1)______
>
> The migration is scheduled to begin on June 22nd and will be completed in three phases over six weeks. During this time, we may experience occasional downtime, ______(2)______ will be kept to a minimum. All departments have been informed of the timeline.
>
> Our DevOps team has already completed the API integration testing. ______(3)______ The cybersecurity team has also confirmed that all data will be encrypted throughout the migration process.
>
> Please ensure that all staff back up their data before June 20th. ______(4)______
>
> If you have any questions, please contact the IT helpdesk at ext. 300.

**Câu hỏi:**

1. (A) The migration has already been completed.
   (B) This upgrade will significantly improve scalability and reduce latency.
   (C) The legacy system has been functioning well for many years.
   (D) We have decided to cancel the migration project.

2. (A) which  (B) who  (C) where  (D) whose

3. (A) However, the results showed several critical errors.
   (B) The results were positive, with no major issues detected.
   (C) We plan to start the testing next month.
   (D) The API has not been tested yet.

4. (A) We appreciate your cooperation and patience during this period.
   (B) The migration was completed successfully last week.
   (C) Please ignore this notice if you are not affected.
   (D) No action is required from department heads.

*Đáp án: 1-B, 2-A, 3-B, 4-A*

**Câu hỏi nhanh (3 câu):**
